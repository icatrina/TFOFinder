Requirements for running TFOFinder on Mac:

1. You need Python 3.x (with pandas, Biopython < 1.79
2. Download and unzip the TFOFinder package from GitHub
3. Open a terminal window and run:
        $ python3 C:/[path for tfofinder_mac]/tfofinder_mac.py
